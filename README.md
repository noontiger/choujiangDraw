# 🎉 幸运抽奖 choujiangDraw

一个**单文件、可移植、离线可用**的在线抽奖网站。无需服务器、无需构建、无外部依赖，**双击 `index.html` 即可在任意浏览器运行**，适合年会、活动、课堂等现场抽奖。

## ✨ 功能

- 🎡 **轮盘抽奖**：Canvas 绘制转盘，支持自定义奖项与权重（权重越大中奖概率越高），带旋转动画与撒花特效。
- 🔢 **数字抽奖**：在指定区间内抽取 1 个或多个数字，支持「不重复抽取」与「结果排序」，带滚动揭示动画。
- 👥 **名单抽奖**：粘贴参与名单（每行 / 逗号 / 空格分隔均可），抽取 1 人或多人，支持「抽中后移出名单」。
- 📤 **导出记录**：每种模式及「全部记录」可一键导出 Excel/CSV（UTF-8 带 BOM，Excel 打开中文不乱码）。
- ⏱ **倒计时开奖**：每个模式可设置开奖前倒计时秒数，全屏「3-2-1-GO」动画后再自动开奖，适合现场烘托气氛。
- 🖥 **大屏投影**：顶部「⛶ 大屏投影」一键进入全屏，放大转盘 / 数字 / 名单结果与中奖展示、隐藏设置面板，按 `Esc` 退出，适合年会 / 活动投屏。
- 🏆 **中奖记录**：每种模式独立记录历史，方便核对。
- 💾 **本地持久化**：所有配置与记录保存在浏览器 `localStorage`，刷新不丢失。
- 🔊 **音效开关**：可选开启 WebAudio 提示音。

## 🚀 使用方式

1. 直接双击 `index.html` 在浏览器打开即可使用；
2. 或部署到任意静态托管（GitHub Pages、Netlify、OSS 等）；
3. 或放到 U 盘 / 内网，随取随用。

## 🗂 项目结构

```
choujiangDraw/

```

## 🔧 技术说明

- 纯原生 HTML / CSS / JavaScript，零依赖、零构建。
- 转盘旋转采用加权随机 + `easeOutCubic` 缓动，结果与实际落点一致。
- 撒花特效、音效均为自实现，无需联网加载任何资源。

> 数据仅保存在你自己的浏览器中，不会上传到任何服务器。
